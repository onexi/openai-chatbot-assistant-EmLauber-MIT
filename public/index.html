How to Use This Webpage

To interact with an AI Assistant, follow these steps:

  1. Click "Select Assistant" to open a popup that lists all assistants available to you.
  2. Click on the assistant name to select it.
  3. Click "Close" to close the popup.
  4. Click "New Thread" to create a new conversation thread with the selected assistant.
  5. Enter your message in the input field.
  6. Click the "Send" button to submit your message.
  7. Wait for the assistant's response, which will appear in the chat window.

If you encounter any issues, please refresh the page or try again later.
